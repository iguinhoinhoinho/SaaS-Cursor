# SaaS Glen Hotel - Frontend

Frontend moderno para o sistema de gerenciamento de hotel SaaS Glen.

## Tecnologias

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

O frontend estará disponível em `http://localhost:5173`

## Build para Produção

```bash
npm run build
```

## Estrutura

- `/src/pages` - Páginas principais (Dashboard, Quartos, Reservas)
- `/src/components` - Componentes reutilizáveis
- `/src/services` - Serviços de API

## Requisitos

- Backend rodando na porta 3000
- Node.js 18+ recomendado
